# TODO Application

A TODO application built with React, Chakra UI, HTML/CSS, Node.js, backed by a RESTful API for managing tasks.

## Features

- Create a new TODO
- Read existing TODOs
- Update TODO details
- Delete a TODO

## Technologies Used

- React: Frontend library for building user interfaces.
- Chakra UI: Component library for React applications.
- RESTful API: Backend service for handling CRUD operations.
